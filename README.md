# Adornofy

## A Pipefy interface clone, with dark theme.

## Instructions
#### -Clone the project
#### -Execute your terminal on the project folder
#### -if you don't have nodeJS on your device, install https://nodejs.org/en/download/
#### -Insert the `npm i`
#### -To run, insert `npm run start`
